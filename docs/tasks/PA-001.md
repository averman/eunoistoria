# Task Spec: PA-001 Scaffold Electron + Vite + React

## 1. Assignment

- **Sub-project:** `packages/power-app`
- **Branch:** `feat/PA-001-electron-vite-scaffold`
- **Depends on:** None
- **Blocked by:** None

## 2. Objective

Scaffold the Power App with Electron + Vite + React. Set up the build tooling, configuration files, and basic window structure so that `pnpm dev` opens an Electron window with a blank React app. This is the foundation for all subsequent Power App work.

## 3. Behavior

- Running `pnpm dev` from `packages/power-app` must start Electron and display a BrowserWindow with React root component loaded.
- Hot module reload (HMR) must work for React components during development.
- Build command (`pnpm build`) must produce optimized Electron app bundle.
- Main process has access to Node.js APIs (file I/O, database, etc.).
- Renderer process is sandboxed: no Node.js imports, communicates via IPC only.
- TypeScript compilation must succeed with strict mode enabled.
- No errors or warnings on startup.

## 4. Interface Changes

- [ ] No interface changes required.

## 5. Test Cases

1. **Dev server starts** — when running `pnpm dev`, Electron window opens without errors.
2. **React renders** — BrowserWindow displays a simple React component (e.g., `<h1>Eunoistoria</h1>`).
3. **HMR works** — editing a React component source file triggers instant reload in the window.
4. **Build succeeds** — `pnpm build` completes without errors, producing dist artifacts.
5. **TypeScript compiles** — no TypeScript errors with strict mode enabled.
